# WattWise AI — frontend

Next.js 14 (App Router) app. See the [repo root README](../README.md) for the full
local setup, and [`docs/ARCHITECTURE.md`](../docs/ARCHITECTURE.md) for how this fits
with the backend.

```bash
npm install
npm run dev       # http://localhost:3000
npm run lint
npm run typecheck
npm run test       # Vitest
npm run test:e2e   # Playwright
```
